Projects Details

Simple Calculator V.2

This beginnier-friendly calculator project performs a series of chained mathematical operations using five user inputs. It first adds the first two numbers, subracts the third, multiplies by the fourth, and divides by the fifth - then prints out each stage of the result, including a cheeky easter egg if the final result exceeds 100.

Built using Python's "try-except" structure to catch input errors, this project taught me the importance of input validation and chaining logic for more complex outcomes. It's simple but mighty - think of it as the first rep at the mental gym.

Techmologies used: Python, CLI, Exception Handling.

Simple Calculator V.2

This project builds upon the first calculator with a more refined understanding of sequential logic flow and Python syntax. The structure mirrors the first version, but this iteration reinforced clean, consistent code habits and focused on improving the readability and maintainability of the logic.

Though similar on the surface, this version helped me cement the routine of writing, testing, debugging, and documenting my code more effectively."

Technologies used: Python, CLI, Error Handeling.

Magic 8-Ball V.1

This project reimagines the playful concept of a Magic 8-Ball using mathematical operations and user input. Instead of randomized responses, it creates a deterministic "fortune" based on the user's chosen numbers. The result not only offers a final number but also includes a humorous celebratory message if the answer is larger than 100 - adding a layer of interactivity and charm.

It helped me merge basic logic with gamification principles to keep users engaged.

Technologies used: Python, User Input, Simple Logic, CLI Interaction.

Magic 8-Ball V.2

This follow-up version takes the same logic structure and fine-tunes it with better comments and cleaner formatting. Think of it like the Director's cut: same script, tighter execution. By iterating the same formula, I solidified my understanding of user-driven input design and streamlined arithmetic logic while playing with humor and interactivity.

This project also reminds me that coding is a muscle - the more you train it, the better your insticts become.

Technologies used: Python, CLI, Exception Handeling, User Interaction.
